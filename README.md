# FlareU-TW-Trans.github.io
# 🐰 FLARE U 台灣翻譯站 | 立于 x 玗進 中文翻譯網站 🦊

本專案是由西卡獨立開發維護的長期存檔平台。旨在為偶像團體提供最高質感、功能完整的繁體中文翻譯影音與社群文章歸檔。系統完全由原生 JavaScript（Vanilla JS）打造，並結合 Firebase 實作即時互動功能。

---

## 🚀 技術棧 (Tech Stack)

*   **前端開發 (Frontend):** HTML5 / CSS3 / JavaScript (Vanilla JS)
*   **資料庫與資料後端 (Backend):** Firebase (Real-time Database)
*   **靜態部署 (Hosting):** GitHub Pages
*   **本地儲存 (Storage):** LocalStorage

---

## ✨ 核心功能亮點 (Key Features)

### 🔍 智慧歸檔與篩選系統
*   **多維度搜尋:** 支援全局關鍵字搜尋、特定月份歸檔查詢。
*   **快速篩選機制:** 支援依出演人員（如：HIGHLIGHT 耀燮&斗俊 等）進行精準快速過濾。
*   **三種版面切換:** 提供三種不同的 UI 視圖模式，滿足使用者不同的瀏覽習慣。

### 📁 六大內容分類區塊
網頁內容依性質嚴謹歸檔至以下六大板塊：
`直播 (Live)` / `節目 (Program)` / `電台 (Radio)` / `媒體 (Media)` / `舞台 (Stage)` / `收藏 (Library)`

### 🎨 互動式影音播放器與卡拉OK動態歌詞
*   **表情牆與即時彈幕系統:** 
    *   每部作品皆串聯 **Firebase Real-time Database** 保存全站表情點擊數據。
    *   首創「即時表情彈幕」，數據即時同步，並開放使用者自由調整表情符號的**大小**、**漂浮速度**與**顯示範圍**。
*   **個人化播放體驗:** 支援字幕大小彈性調整、原生全螢幕模式，完美適應手機、平板與電腦多端裝置。
*   **動態歌詞應援模組:** 創新研發卡拉OK式動態滾動歌詞介面，大幅降低粉絲練習應援口號的門檻。

### 💾 雙層數據持久化與排行榜機制
*   **自定義愛心收藏:** 卡片式組件整合 **LocalStorage**，使用者可自由收藏喜愛影片至個人專屬的「收藏欄位」。
*   **舞台推薦與全站排行榜:** 整合 LocalStorage 本地收藏與 Firebase 全體數據，即時連動並渲染出全站的「熱門舞台排行榜」。

### 📰 複合式動態排版
*   打破單一影音平台限制，針對特定企劃（如：雜誌拍攝幕後），將「社群文章翻譯」與「影片翻譯」完美融合，依內容不時動態變更版面樣式。
